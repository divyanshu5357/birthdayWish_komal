# Birthday Celebration

A React + Vite birthday greeting experience with password gate, confetti, glowing message reveal, candle animation, and looping celebration flow.

## Setup

```bash
npm install
npm run dev
```

## Notes

- Update `src/config.js` to change the password, names, and message text.
- Add optional background music at `public/music/bg-music.mp3` if you want audio playback.
